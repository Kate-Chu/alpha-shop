## 本地安裝流程

1. 請確認電腦已經安裝 Node.js 與 npm

2. 打開終端機，輸入以下指令將此專案 clone 到本地

```
git clone https://github.com/Kate-Chu/alpha-shop.git
```

3. 終端機移動至專案資料夾，輸入指令安裝套件

```
cd 專案資料夾
npm install
```

4. 輸入以下內容，開始運行 React

```
npm run start
```

5. 結束運行請在終端機按下 ctrl + c 即可終止程式。
